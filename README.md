CMS Seiten in Banner einfügen
====

### Shopversion
OXID eShop 6


### Installation

`composer require --no-scripts --update-no-dev --no-interaction --optimize-autoloader swinde/cms2banner`

Mit diesem Modul ist es möglich kurze Texte in den Flexslider einzufügen.
